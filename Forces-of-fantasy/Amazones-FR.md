# **AMAZONES — LISTE D’ARMÉE LÉGACY**
# =====================================================================

## **INTRODUCTION**

Ce document permet d’utiliser une armée **Amazones** dans *Warhammer: The Old World* via une liste communautaire non‑officielle.  
Il reprend les figurines de Warhammer Fantasy Battles et les adapte au nouveau système.

Ces règles :
- ne sont **pas destinées au jeu égal**,  
- ne seront **pas mises à jour** suivant les sorties officielles,  
- permettent de jouer une faction ancienne avec un style compatible TOW.

Les Amazones, créées par les Slanns, ont hérité de fragments de leur technologie ancienne. Leur société parthénogénétique, isolée au cœur de Lustrie, repose sur des traditions guerrières, magiques et scientifiques préservées depuis l’Âge des Anciens.

## **RÈGLES SPÉCIALES**

### **Matriarch Authority**
Les unités de Base Amazones (Guerrières Tribales, Anakonda, Colibris) à **12"** peuvent utiliser son **Cd**.  
Une fois par bataille :  
→ *Sister’s Command* : toutes les unités amazones dans **18"** gagnent **+1 Résultat de Combat** pour le tour.

### **Ancient Wisdom**
- Si l’armée en inclut une : relance de **1 dé** pour le jet de déploiement.  
- Si elle est Général : déploiement d’une **unité de Base après l’adversaire** (avant Éclaireurs).
